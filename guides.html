How to Install

Follow these simple steps to get started on Android or Windows.

android

Android Installation

  1. Download the .APK file from the Home page.
  2. Open the file from your notification bar or File Manager.
  3. If prompted, go to Settings and enable "Install from Unknown Sources".
  4. Tap Install and wait for it to finish.
  5. Open the app and create your account!
desktop_windows

Windows Installation

  1. Download the .EXE installer.
  2. Double-click to run the setup.
  3. Note: Since we are an indie store, Windows Defender might show a blue warning.
    Click "More Info" → "Run Anyway".
  4. Follow the setup wizard to complete installation.
