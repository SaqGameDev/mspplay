Installation Guide

Follow these simple steps to install Gaza Store on your device.

android

Android

  1. Download the .APK file from the Home page.
  2. Tap the notification to open the file.
  3. If prompted, go to Settings and allow "Install from Unknown Sources".
  4. Tap Install and wait for the process to finish.
  5. Open the app and create your account!

desktop_windows

Windows

  1. Download the .EXE installer.
  2. Double-click the file to run the setup.
  3. Note: Windows Defender might show a blue warning screen.
    Click "More Info" → "Run Anyway".
  4. Follow the setup wizard to complete installation.
